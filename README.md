# Arapaima Marketplace (Amazonia IA)

**Arapaima Marketplace** (también conocido como **Amazonia IA**) es una plataforma integral diseñada para empoderar a las comunidades indígenas de la Amazonía. Combina un marketplace de comercio justo con herramientas avanzadas de inteligencia artificial para la preservación cultural y ambiental.

## 🌟 Propósito
El proyecto busca impulsar el desarrollo económico sostenible de los pueblos originarios, permitiendo que su sabiduría y productos (artesanías, medicinas naturales, etc.) lleguen al mundo sin intermediarios, derribando barreras lingüísticas y protegiendo la biodiversidad regional.

## 🚀 Características Principales

- **Marketplace Responsable:** Canal de venta directo para productores indígenas, eliminando intermediarios y garantizando precios justos.
- **IA Visual Amazónica:** Modelos de visión por computador entrenados para reconocer especies de flora y fauna endémicas.
- **Preservación Lingüística:** Traductores neuronales de texto y audio para lenguas nativas (como el Jivi), facilitando el comercio y la comunicación global.
- **Impacto Económico y Cultural:** Generación de ingresos sostenibles y documentación activa de lenguas y conocimientos ancestrales.

## 🛠️ Stack Tecnológico

- **Frontend:** [React](https://reactjs.org/) con [Vite](https://vitejs.dev/)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [Radix UI](https://www.radix-ui.com/), [Lucide React](https://lucide.dev/)
- **Base de Datos & Auth:** [Supabase](https://supabase.com/)
- **Estado & Datos:** [TanStack Query](https://tanstack.com/query/latest)
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/) (vía Tailwind)
- **Gráficos:** [Recharts](https://recharts.org/)

## 💻 Ejecución del Proyecto

1. **Instalar dependencias:**
   ```bash
   npm i
   ```

2. **Iniciar servidor de desarrollo:**
   ```bash
   npm run dev
   ```

3. **Construir para producción:**
   ```bash
   npm run build
   ```
